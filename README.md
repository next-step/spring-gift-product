# spring-gift-product

## 상품 API

### 1. 상품 생성

- URL : /api/products
- 메서드 : POST
- 기능 : 상품 생성
- 설명 : 새 상품을 등록한다.

### 2. 상품 조회

- URL : /api/products/{productId}
- 메서드 : GET
- 기능 : 상품 조회
- 설명 : 특정 상품의 정보를 조회한다.

### 3. 상품 수정

- URL : /api/products/{productId}
- 메서드 : PUT
- 기능 : 상품 수정
- 설명 : 기존 상품의 정보를 수정한다.

### 4. 상품 삭제

- URL : /api/products/{productId}
- 메서드 : DELETE
- 기능 : 상품 삭제
- 설명 : 특정 상품을 삭제한다.
