# spring-gift-product


# 구현 기능
| URL                     | Method | Description              |
|-------------------------|--------|--------------------------|
| `/api/products`         | GET    |  제품 전체를 조회
| `/api/products`         | POST   |  제품을 추가     |
| `/api/products/{id}`    | PUT    |  특정 상품의 내용을 수정|
| `/api/products/{id}`    | DELETE |  특정 상품 삭제     |
