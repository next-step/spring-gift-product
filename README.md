# spring-gift-product

| URL                     | Method | Description              |
|-------------------------|--------|--------------------------|
| `/api/products/{id}`    | GET    | (상품 조회) 특정 상품의 정보를 조회한다. |
| `/api/products`         | POST   | (상품 추가) 새 상품을 추가한다.      |
| `/api/products/{id}`    | PUT    | (상품 수정) 특정 상품의 정보를 수정한다. |
| `/api/products/{id}`    | DELETE | (상품 삭제) 특정 상품을 삭제한다.     |
