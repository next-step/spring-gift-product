# spring-gift-product

## 1단계 - 상품 API
### 구현할 기능 목록
1. 상품 추가 - `POST /products`
2. 상품 수정 - `PATCH /products/{id}`
3. 상품 삭제 - `DELETE /products/{id}`
4. 전체 상품 조회 - `GET /products`
5. 특정 상품 조회 - `GET /products/{id}`
