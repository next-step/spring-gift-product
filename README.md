# spring-gift-product

### 작성할 상품 API 명세서 (250624)
| URL | 메서드 | 기능 | 설명 |
| :--- | :--- | :--- | :--- |
| /api/products | POST | 상품 생성 | 새 상품을 등록한다. |
| /api/products/{productId} | GET | 상품 조회 | 특정 상품의 정보를 조회한다. |
| /api/products/{productId} | PUT | 상품 수정 | 기존 상품의 정보를 수정한다. |
| /api/products/{productId} | DELETE | 상품 삭제 | 특정 상품을 삭제한다. |
